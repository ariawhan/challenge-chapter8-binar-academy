## Run Application

This application use node JS for runtime environment

To install all package modules use :

```
& npm install

```

To install database :

```
& npm db:create
& npm db:migrate
& npm db:seed

```

To run application use :

```
& npm run start
& npm run develop

```

To testing application use :

```
& npm run test

```

To eslint check application use :

```
& npm run lint

```
